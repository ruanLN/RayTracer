Bem-vindo a página do trabalho final de FCG, 2015/1

Na disciplina de Fundamentos de Computação Gráfica(FCG) do semestre 2015/1, foram ofertadas duas propostas de trabalho final, uma delas, a implementação de um jogo, Ludo Runner, e outra, a implementação de um Ray Tracer para cenas estáticas.

Instruções de compilação

Faça clone/checkout da última versão estável (master) ou de desenvolvimento(develop, pode, e provavelmente conterá bugs). Para isto use os comandos:

$ git clone https://github.com/ruanLN/RayTracer.git
$ git checkout origin/master #<- ou develop, no caso da versão em desenvolvimento
$ git checkout -b master #<- ou develop, no caso da versão em desenvolvimento

O ultimo passo não é necessário, ele apenas cria um branch local para evitar de manter o repositório no estado "HEAD_FETCH", que pode ocasionar perda de dados. Atualmente para compilar é necessário abrir o projeto na pasta RayTracer/RayTracerNonQt utilizando o Qt Creator e dar build, em futuras versões planeja-se a adição de um Makefile.

Imagens geradas:

Imagens com supersampling de 1(normal), 2, 4 e 8 (são lançados n^2 raios por pixel)

Imagem utilizando "Normal Coloring", que utiliza as normais da face para calcular sua cor (Sugerido em http://www.cs.rug.nl/svcg/ComputerGraphics/Raytracer)

6 planos, 1 esfera, 4 raios por pixel, 800x600, renderizado em 107 segundos em um Intel(R) Dual Core T4400 2.2GHz, programa single thread.

Primitivas implementadas

  1. Esferas
  2. Triângulos

Descrição do arquivo de entrada

O arquivo de entrada tem o formato YAML e foi baseado no arquivo do RayTracer de Gronigen, junto com o codigo existe um arquivo exemplo com todos comandos possíveis
